Fábrica de Conteúdo

Transforma arquivos .md em carrosséis e posts estáticos prontos pra download.

Carrosséis

Coloque seus arquivos em content/carrossel/ com front matter mínimo e eles aparecem aqui automaticamente.

    {% assign carrossel_pages = site.pages | where:"type","carrossel" %} {% for p in carrossel_pages %}
  • {{ p.title | default: p.name | replace: '.md','' }}
    • {% else %}
  • Nenhum carrossel ainda.
    • {% endfor %}

Posts estáticos

Arquivos em content/static/ viram cards de post único.

    {% assign static_pages = site.pages | where:"type","static" %} {% for p in static_pages %}
  • {{ p.title | default: p.name | replace: '.md','' }}
    • {% else %}
  • Nenhum post estático ainda.
    • {% endfor %}
