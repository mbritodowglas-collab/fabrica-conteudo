← Voltar para a Fábrica

Como funciona este modelo

Aqui você gera apenas o roteiro em .md com espaço reservado para a mídia. Depois, a imagem ou o vídeo é encaixado no Canva/CapCut.

  • Cada slide tem midia: true para reservar espaço visual.
  • O texto fica mais compacto para não brigar com a imagem/vídeo.
  • Você exporta os slides aqui e finaliza o design no editor de mídia.
