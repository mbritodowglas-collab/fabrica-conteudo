# Fábrica de Conteúdo

Ferramenta web minimalista para organizar arquivos `.md` em blocos de texto prontos para redes sociais.

Roda 100% estático em **GitHub Pages**.  
Não gera imagem, não exporta PDF — é só texto, limpo e copiável.

---

## Estrutura

```text
/
├── index.html
├── /modes
│   ├── carrossel.html
│   └── static.html
├── /assets
│   ├── /css/style.css
│   ├── /js/carrossel.js
│   └── /js/static.js
└── README.md
